# 🐦 Twitter N1 — Backend

Este é o backend do projeto **Twitter N1**, uma aplicação estilo “mini Twitter” desenvolvida com **Node.js**, **Express** e **MongoDB**.

## 🚀 Tecnologias

- Node.js
- Express
- MongoDB Atlas (via Mongoose)
- Dotenv
- Nodemon (para ambiente de desenvolvimento)
- CORS e Body-Parser
